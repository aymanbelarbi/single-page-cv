# Single-Page CV

A semantic HTML CV showcasing career history, education, and skills - built as a solution to the roadmap.sh [Single-Page CV](https://roadmap.sh/projects/single-page-cv) project.

## Project Overview

This project demonstrates foundational HTML skills by creating a structured, single-page curriculum vitae using semantic HTML5. The focus is on clean markup, accessibility, and SEO optimization - with styling to be added in future iterations.

## Requirements Met

- **Semantic HTML Structure** - Proper use of `<header>`, `<main>`, `<section>`, `<article>`, and `<footer>` elements
- **SEO Meta Tags** - Description, keywords, and robots meta tags included
- **Open Graph Tags** - OG tags configured for enhanced social media sharing
- **Favicon** - Custom favicon linked in the document head
- **Single-Page Layout** - Clean sections for professional summary, skills, education, and online profiles

## Technologies Used

- **HTML5** - Semantic markup
- No CSS or JavaScript (minimal approach focusing on structure)

## Local Development

Clone the repository and open the HTML file in your browser:

```bash
git clone https://github.com/aymanbelarbi/single-page-cv.git
cd single-page-cv
open index.html
```

## CV Sections

- **Header** - Name, title, and contact information
- **Professional Summary** - Career overview and objectives
- **Skills** - Technical and soft skills
- **Education** - Academic background
- **Online Profiles** - LinkedIn and GitHub links

## Learning Outcomes

This project provided hands-on experience with:

- Writing semantic, accessible HTML
- Implementing SEO best practices
- Structuring content for readability and maintainability
- Preparing markup for future CSS styling
- Adding meta tags for search engines and social media
